# xv6_thread_implementation

xv6上でthreadを実装する．

## 基本設計
スレッドの特徴は，生成元のプロセスとアドレス空間を共有し，スタック領域だけ独自のものを持つことである．

本実装では，既存のfork関数を改良し，新たなプロセスのページテーブル割り当ての処理を改良することでスレッドを実装した．
